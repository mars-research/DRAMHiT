compute/memory.

Compute are synchronous, 3 - 4 cycles.
Memory are asynchronous, depends data locality and system bandwidth and system load factor, 3 - 200 cycles.

we treat cpu and memory as 2 separate network devices or NIC, where they can transmit and receive information.

is saturated (recv bandwidth), system throughput will remain bottlenecked. Also any OOO cpu would continue processing
